I built an audio VST, compatible with most major Digital Audio Workstations for equalizing audio. It's based on a tutorial from "Free Code Camp" on youtube: http://youtube.com/watch?v=i_Iq4_Kd7Rc

Although this piece of software is relatively simple, working through this project gave me a view into how to use the JUCE framework to manipulate an audio signal. There were many key fundamental aspects of the JUCE framework that I learned through this project: the most important aspects of this project included learning how to create the "processBlock()" function to actually process an audio signal, as well as managing the state of all of the knobs with the "audioProcessorValueTreeState."

In theory, several instances of this VST can be combined within an Audio VST, making it surprisingly robust in its application despite its simplicity.

Leaving this up to show my progress, this is an earlier project. 

This is a simple pinball game with a 2d physics engine. I built this while I was studying at CU boulder to practice Javascript, as most of the degree used Python, C++, and SQL and I felt that Javascript would be important to know as well. 

It simply works by running a game loop on a javascript canvas, and running a collision detector at each frame. Event listeners pick up the left and right arrows, and handlers take care of the response. Easy peasy videogame.
